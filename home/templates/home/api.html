How to Use Our API


HTTP Method:

POST

Parameters:

stock_characteristic_file - CSV file for stock characteristics
stock_price_data_file - CSV file for stock price data
file_key - 12 character value that represents the files that have been uploaded previously. This means you do not have to uplaod the same files multiple times. Example: YVLRBOEFEBPA
lower_window - integer value that is a lower bound of the results to be shown. Example: -10
pper_window - integer value that is an upper bound of the results to be shown. Example: 10

All of these parameters are added to the end of the webpage's URL. Without the two files or a file key, no results will be produced.

Example Input:

https://seng3011-eventstudy.herokuapp.com/eventapi/?file_key=YVLRBOEFEBPA&upper_window=10&lower_window=-10
